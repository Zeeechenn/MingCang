# M58 Phase 0.5 — 开源量化模型候选调研(2026-07-03)

> 结论先行:owner「Qlib 系近年不太行」部分成立,但归因要精确——不是 Qlib 基础设施失效,
> 而是其 2021-22 代模型库(LGBM/HIST/IGMTF/ADD)近三年缺乏可信 A 股公开实证,官方自认
> concept drift 未解;明仓 M26/M27 两轮负结果与大趋势一致、非孤例。开源圈**没有**免验证
> 可用的近年 A 股 alpha 模型;有 2 个值得小成本预实验的新方向(RD-Agent(Q)、MASTER)。

## 分档

- **A(直接入 Phase 1 网格)**:无——没有候选同时满足"近年强实证+第三方独立复现"。
- **B(小成本预实验)**:
  - **RD-Agent(Q)**(微软,NeurIPS 2025):LLM 多智能体自动因子挖掘+模型联合优化,基于 Qlib 底座;CSI500 IC=0.0288/ICIR=0.183(2024-25 窗口),号称年化≈Alpha158 基线 2 倍、因子数省 70%;无重 GPU 需求但要 LLM API(论文用 o3-mini,单轮 <$10),接入 5-8 天。与明仓 LLM 方法论契合;未被明仓评估过。
  - **MASTER**(上交,AAAI 2024):横截面 Transformer(momentary+cross-time 股间相关+市场信息引导特征选择);多篇 2025 后续论文以其为强基线;需 GPU 训练;须用作者独立仓库(第三方 Qlib 版本有出入);无明仓生态外独立复现实证,须本地验证真实 IC。
- **C(不做)**:
  - Qlib 传统模型库原样重测——明仓 M26/M27 已证伪(IC 转负/分层非单调),无版本级变化;
  - Kronos 原样重测——AAAI 2026 录用但权重仍是 2025-08 checkpoint(GitHub 活跃在应用层),明仓 M26.2 微调实测 IC=-0.047/ICIR=-0.196,无版本级变化;
  - VeighNa/vnpy——实盘执行框架(4.0 有 vnpy.alpha 新模块但非验证过的 alpha 模型),对明仓价值=券商下单接口,与 alpha 调研目标不匹配;
  - FinRL 系——交易执行 RL 框架非 alpha 信号源,A 股实证教学级;
  - AlphaForge——基线对比 IC=0.0146 弱于上两者,独立实证少,留档观察。

## 纪律

论文数字=学术设定样本内结果,一律不采信;任何 B 档候选过明仓 promotion gate
(DSR/PBO/trial-count + 前向)之前只是研究玩具。预实验注册进 m29 registry 记 trial。
