## 北极星

- 目标：真实帮助 owner 实盘炒股、尽可能提高收益；回撤容忍约 20%。
- 最终产出物：**每日盘后操作面板**（买入候选 + 持仓体检 + 避雷警示 + 复盘归因），所有模块的输出向它汇聚。

## M58 决策公式重构【核心线 / 2026-07-03 立项】

Goal: 每个决策功能位用自己最有效的公式，彻底替代单一综合分。

证据基础（2026-07-03 诊断，详见 `~/.stock-sage/gate_b.db` 与当日分析）：
- 现生产综合分（0.6技术+0.4标题情感）7 周窗口横截面 ICIR ≈ **-0.03**（零区分力）；
- 动量分认弱股准（bottom20% 5日 -2.05pp）但挑强股弱 → 天然是**避雷器**；
- 6 月动量 ICIR 2.59 系单边行情 7 天幻觉（时段拆解证实），故一切结论必须跨 regime；
- ATR 出场：M21.4 复盘已证伪"初始止损过窄"假设（v1 初始 ×2.0 零触发，亏损来自跳空滑价），出场问题=trailing 倍数敏感性 + 滑价成本分布。

信号族范围（owner 2026-07-03 补充裁决）：**技术 / 动量 / 量化 / 新闻 四族全测**。量化不因近期负面证据简单抛弃——其成因复杂（06-06 模型断供事故导致近月"量化分"实为动量；LGBM 服役期仅 18 天单一窗口；M26/M43 证伪针对的是特定构造非量化整族）。量化族候选 = LGBM 重训基线 + **外部开源模型调研**（Qlib 系近年实证、清华 Kronos——注意 repo 内已有 M26/M27 Kronos 评估与微调产物、VeighNa 等，先出候选清单+接入成本再决定入网格哪些）。

Phases：
0.5 **外部开源量化模型调研（并行先行）**：产出候选清单（信号形态/可本地跑/数据需求/近年公开实证/接入成本/与 repo 既有评估的关系），owner 圈选后入 Phase 1 网格。
1. **价格系长回测网格（先行，零 LLM）**：技术 × 动量 × 量化（LGBM 基线 + 0.5 圈选的外部候选）的全部组合 + 出场参数（trailing ×2.0~×3.5、浮动止盈变体）在 **1490 天含 holdout** 的大样本通道上扫（复用 `exit_sweep`/`backtrader_eval`，M21.4 裁决通道并入此处）；按功能位分别评估（选股用头部区分力、避雷用尾部区分力、出场用净收益/回撤比）；DSR/PBO/trial-count 全程记账（M51 D1 合约）；回撤约束 ~20% 作为参数筛选硬条件。
2. **三假设预注册前向影子（4-6 周，跨 regime）**：①避雷线=动量末档→未来5日显著弱；②选强线=技术头档（+M54 v2 新闻触发，等其裁决后并入）；③出场线=Phase 1 胜出的 trailing/滑价参数。注册进 `m29_hypothesis_registry`。
3. **按功能位定稿 + 灰度进 M59 面板**：每个功能位给"公式 + 证据 + 适用边界"，用户确认后替换面板对应区块的填充规则。

新闻组件的位置：分轨处理——价格系不等新闻；M54 裁决后 v2 新闻作为**选强线增强项**并入 Phase 2/3（新闻层优化不是无用功，但它只增强有正文覆盖的股票，见 M54 §13 适用边界）。

Stop conditions: 单窗口/单 regime 结果不得定稿；每个功能位的公式必须报告同池等权基线超额；试验次数必须记账；不改生产行为直到用户逐项确认。
